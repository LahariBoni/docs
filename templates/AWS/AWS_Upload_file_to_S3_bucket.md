### Connect to AWS


```python
s3 = boto3.client('s3',

                  aws_secret_access_key=SECRET_ACCESS_KEY)
```

## Output

### Upload data


```python
with open(BUCKET_OBJECT_KEY, "rb") as f:
    s3.upload_fileobj(f, BUCKET_NAME, BUCKET_OBJECT_KEY)
```
